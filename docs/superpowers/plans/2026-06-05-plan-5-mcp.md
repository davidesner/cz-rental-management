# Plan 5 — MCP server + skill template + <property-name-a> MCP replay

**Goal:** Stdio MCP server (fastmcp) exposing REST endpoints as MCP tools. Local Claude Code can drive the whole system via tools. End: replay <property-name-a> 2024 scenario via MCP and verify +693 Kč.

## Architecture

- `mcp/index.ts` — fastmcp server entry; reads `RENTAL_API_URL` and `RENTAL_API_TOKEN` from env.
- `mcp/client.ts` — `fetch`-based REST client with Bearer auth, throws on non-2xx with details.
- `mcp/tools/*.ts` — one file per domain area, each exporting `addXxxTools(server, client)` that registers tools.

Tools mirror REST endpoints. Naming: `<resource>_<action>` (e.g. `payments_record_batch`, `reconciliations_compute`).
